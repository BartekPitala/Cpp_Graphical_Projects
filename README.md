# Cpp_Graphical_Projects

Repository containing my projects related to 2D graphics programming in C++.

# OwnPaint:
A basic implementation of MS Windows Paint app written in SFML library. For details, see ReadMe in the project's main catalog.

# Acceleration Field:
Application that displays vectors of the Earth's acceleration field on a 2D map, and animates the movement of a ball from the highest point of the map, accordingly to the counted highest drop vector. For details, see ReadMe in the project's main catalog.
